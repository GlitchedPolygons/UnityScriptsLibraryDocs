Class Relay

Logic relay that can be fired with or without a delay.

When fired, a relay will invoke all of its registered actions at once.

You can decide what happens on fire by subscribing/unsubscribing listeners to the Fired event (via AddListener(Action) and RemoveListener(Action)) or by hooking the events up directly in the Relay's inspector.

Fire()

Fires the relay, invoking all registered method calls.

This waits for GlitchedPolygons.Logic.Relay.delay seconds and then calls GlitchedPolygons.Logic.Relay.OnFire, thus triggering the Fired and GlitchedPolygons.Logic.Relay.onFire events.
