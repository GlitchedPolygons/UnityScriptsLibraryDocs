Class GlitchEd25519

C# translation of the GlitchEd25519 fork of the orlp/ed25519 C library.

Exposes functions for generating Ed25519 key-pairs, performing key exchange operations on them, as well as signing and verifying messages.

AddScalar(ref Span<Byte>, ref Span<Byte>, ReadOnlySpan<Byte>)

Adds scalar to the given key pair where scalar is a 32 byte buffer (possibly generated with ed25519_create_seed), generating a new key pair.

You can calculate the public key sum without knowing the private key and vice versa by passing in NULL for the key you don't know.

This is useful for enforcing randomness on a key pair by a third party while only knowing the public key, among other things.

Warning: the last bit of the scalar is ignored - if comparing scalars, make sure to clear it with scalar[31] &= 127.
